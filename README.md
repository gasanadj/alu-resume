# Online Resume
This is a personal resume that covers the summative

## Usage
- Clone the repo `git clone https://github.com/gasanadj/alu-resume`
- Change the directory to the cloned repo `alu-resume`
- Open the file `index.html`

## Sections
- Header
- About Me
- Experience
- Mission
- Skills
- Projects
- Contacts

## Tech Stack
- HTML
- CSS
- JS
